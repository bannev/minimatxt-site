Frequently Asked Questions

  • Does MinimaTXT sync with iCloud?
    No. All files are stored locally on your iPhone. The app does not use iCloud, servers, or external storage.
  • Can I import .txt files?
    Yes. Tap the menu in the top-right corner and choose "Add from Files" to open files from the Files app.
  • Can I export my files?
    Yes. You can export individual files, multiple selections, or use "Export All as ZIP" to save everything in one archive.
  • Does MinimaTXT support Markdown?
    No. MinimaTXT is intentionally focused on plain .txt files only — no formatting, no markup.
  • The app crashed — what should I do?
    Please send us an email with as much detail as possible (what you were doing, file name, etc.). We'll investigate promptly.
  • I accidentally deleted a file. Can you help recover it?
    No. All files are stored locally on your device. MinimaTXT does not back up your data or sync it with any servers. Not even Apple has access to your content. We recommend using "Export All as ZIP" regularly to keep external backups.
  • Does MinimaTXT work offline?
    Yes. The app is fully offline. It does not require an internet connection and does not communicate with any servers.
  • Can I export all my files at once?
    Yes. Tap the menu and choose "Export All as ZIP" to create a compressed archive of all your files. You can save or share it via the Files app, AirDrop, or other apps.
  • Are there any restrictions on file names?
    Yes. To avoid issues with other systems, file names must follow these rules:
    – Cannot be empty
    – Cannot start with a dot (.) or ellipsis (...)
    – Cannot contain characters like: \ / : * ? " < > |
  • What happens if I import a file with a name that already exists?
    MinimaTXT automatically appends a number like (1), (2), etc., to avoid overwriting your existing file. This happens silently, without asking for confirmation.
  • How do I delete a file?
    Swipe left on a file in the file list, then tap "Delete".
  • How does saving work?
    All changes are saved automatically:
    – every 30 seconds while editing
    – when the app goes into the background
    – when you close the editor
    There’s no need to save manually.
  • Can I organize files into folders?
    Not yet. All files are listed in a single view. Folder support is planned for a future update.
  • Can I change fonts or enable dark mode?
    MinimaTXT uses a simple, clean theme with one default font. Dark mode is supported and switches automatically based on your iPhone’s appearance settings (under Settings → Display & Brightness).
  • Can I use MinimaTXT on an iPad?
    We DO NOT recommend using the app on an iPad. It has been designed and tested specifically for iPhone, so the user experience on iPad may and will be poor. Once the official iPad version is released, it will be available on the App Store for iPad.
